🎮 ClickerVibe - Community Game
Um jogo multiplayer em tempo real desenvolvido para demonstrar conceitos fundamentais de Python, WebSockets e Interatividade em sala de aula. O objetivo é simples: todos os alunos clicam juntos para aumentar a "Energia da Comunidade".

🚀 Objetivo Pedagógico
Este projeto foi pensado como uma ferramenta didática para introduzir crianças ao mundo da programação, abordando:

Eventos de Clique: Como o computador reage à ação do usuário.

Variáveis Globais: Como armazenamos uma pontuação que todos compartilham.

Comunicação em Tempo Real: Introdução ao conceito de como jogos multiplayer (como Roblox ou Minecraft) funcionam por trás das cenas.

🛠️ Tecnologias Utilizadas
Backend: Python com Flask.

Real-time: Flask-SocketIO (WebSockets).

Frontend: HTML5, CSS3 (Bootstrap) e JavaScript.

📦 Como rodar o projeto
Instale as dependências:

Bash
pip install flask flask-socketio
Inicie o servidor:

Bash
python app.py
Acesse no navegador:
Abra http://localhost:5000
